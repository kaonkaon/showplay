### Installation
* clone the git
```
git clone https://github.com/kaonkaon/showplay
```
* enter showplay dir
```
cd showplay
```
* give it permission to run
```
chmod +x showplay
```
* move it to bin dirrectory
```
sudo cp showplay /usr/local/bin/
```
